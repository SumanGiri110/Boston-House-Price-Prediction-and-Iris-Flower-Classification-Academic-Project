## Machine Learning Projects: Iris Classification & Boston House Price Prediction
This repository contains two beginner-friendly machine learning projects implemented using Python and Scikit-Learn:
- Iris Flower Classification using Decision Tree
- Boston House Prices Prediction using Regression <br>
These projects demonstrate basic machine learning workflows including data loading, preprocessing, model training, evaluation, and prediction.
# Iris Flower Classification
## Overview
The Iris dataset is a classic dataset used in machine learning for classification tasks. The goal is to classify iris flowers into one of three species:
- Setosa
- Versicolor
- Virginica <br>
This project uses a Decision Tree Classifier to predict the species based on flower measurements.
## Dataset Features
The dataset contains the following features:
- Sepal Length

- Sepal Width

- Petal Length

- Petal Width

## Target:

- Flower Species

## Model Used

- Decision Tree Classifier from Scikit-Learn
  - Why Decision Trees?
    - Easy to interpret
    - Works well with small datasets
    - Handles nonlinear relationships

# Boston House Price Prediction
This project predicts housing prices in Boston using regression techniques based on several economic and housing features.
## Dataset Features
Some important features include:

- CRIM – Crime rate

- RM – Average number of rooms per house

- AGE – Proportion of old houses

- TAX – Property tax rate

- PTRATIO – Student-teacher ratio

- LSTAT – Lower status population percentage

## Target:

- MEDV – Median house value

## Model Used

- Linear Regression from Scikit-Learn

  - Why Linear Regression?

    - Simple baseline model

    - Easy to interpret

    - Works well for continuous prediction problems

## Results

- Iris Classification achieves high accuracy.

- Boston Housing model predicts house prices based on multiple socio-economic features.
